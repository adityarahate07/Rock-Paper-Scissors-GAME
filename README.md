# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Play against the computer and test your luck!

---

## 🚀 Features

* 🪨 Rock, 📄 Paper, ✂️ Scissors gameplay
* 🤖 Random computer choice generation
* 🧠 Game logic to determine winner
* 📊 Live score tracking (User vs Computer)
* 🎨 Interactive UI with hover effects
* 🟢 Dynamic result messages (Win / Lose / Draw)

## ⚙️ How It Works

1. User clicks on any choice (Rock, Paper, or Scissors)
2. Computer generates a random choice
3. Game compares both choices:

   * Rock beats Scissors
   * Paper beats Rock
   * Scissors beats Paper
4. Result is displayed instantly
5. Score updates automatically

---

## ▶️ How to Run

1. Download or clone this repository
2. Open the project folder
3. Run `index.html` in your browser
